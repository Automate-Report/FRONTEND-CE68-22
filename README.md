# FRONTEND-CE68-22
This repository contains the code for the frontend of the CE68 project. The frontend is built using React and is designed to provide a user-friendly interface for interacting with the backend services.
## Getting Started
To get started with the frontend, copy these to terminal:
1. Clone the repository:
   ```bash
   git clone {repository_url}
   ```
2. Navigate to the project directory and run:
   ```bash
    cd .\frontend-ce68-22\
    npm install
    npm run dev
    ```
